🏠 House Price Prediction using Linear Regression
📌 Overview

This project implements a Machine Learning model using Linear Regression to predict house prices based on features like area, number of bedrooms, and age of the house.

The project demonstrates the complete ML workflow including:

Data loading
Data preprocessing
Model training
Prediction
Model evaluation
🎯 Objective

To build a regression model that predicts house prices accurately using input features:

Area 📐
Bedrooms 🛏️
Age 🏡
🛠️ Technologies Used
Python 🐍
Pandas
Scikit-learn
